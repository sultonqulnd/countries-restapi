You can use countries-restapi.vercel.app as endpoint for using https://restcountries.com.

This api has extra facilities like pagination, sorting and total .
1. countries-restapi.vercel.app/all?page=1&limit=5 - getting initial 5 countries, default limit value equals to 10.
2. countries-restapi.vercel.app/all?sort=title&order=desc - getting countries in descending order by title which is the common name of a country. Order values are asc or desc. You can use different properties for sorting in the future, but now only one property, title is supported.
